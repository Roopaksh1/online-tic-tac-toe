## Installation

- Clone the Repo

```
git clone git@github.com:Roopaksh1/online-tic-tac-toe.git
```

- Install the dependencies

```
cd client
npm install

cd server
npm install
```

- Run the server

```
cd server
node index.js
```

- Run the client

```
cd client
npm start
```
